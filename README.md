# 🗂️ App Catálogo de Productos

> Aplicación de escritorio para la gestión y visualización de un catálogo de productos, desarrollada en C# con arquitectura en capas.

## 📋 Descripción

Aplicación de escritorio que permite gestionar un catálogo de productos de forma estructurada. Desarrollada aplicando principios de **arquitectura en capas** para separar responsabilidades y facilitar el mantenimiento del código.

---

## 🏗️ Arquitectura en Capas

```
📁 AppCatalogoProductos
├── 📂 Dominio          → Entidades y modelos del negocio
├── 📂 Negocio          → Lógica de negocio y reglas de la aplicación
└── 📂 Presentacion     → Interfaz de usuario y formularios
```

Cada capa tiene una responsabilidad única y se comunica únicamente con la capa adyacente, respetando el principio de **separación de responsabilidades**.

---

## ✅ Funcionalidades

- Listado de productos del catálogo
- Carga y edición de productos

---

## 🛠️ Tecnologías

- **Lenguaje:** C#
- **Plataforma:** .NET Framework 4.8
- **Patrón:** Arquitectura en capas (Dominio · Negocio · Presentación)
